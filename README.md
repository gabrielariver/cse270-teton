# 🌐 Teton Idaho Chamber Website (CSE 270 Project)

## 🚀 Description
This project is a fully developed **Chamber of Commerce website** built as part of the **CSE 270 course**.  
It focuses on delivering a functional, user-friendly web experience while incorporating **automated testing and CI/CD practices**.

🔗 Live site:  
https://gabrielariver.github.io/cse270-teton/

---

## 🧠 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Git & GitHub
- GitHub Pages (deployment)

### 🔧 Testing & Automation
- Selenium (UI automation)
- Pytest (test framework)
- GitHub Actions (CI/CD workflows)

---

## 📌 Key Features
- Responsive Chamber of Commerce website
- Business directory and membership functionality
- Join form with validation and submission flow
- Structured navigation and user-friendly layout
- Automated smoke testing using Selenium
- Continuous Integration with GitHub Actions
